Testimonial

This is ML web which is using a Machine Learning model in backed for prediction.
The motivation of developing this website is to predict the price of the car more accurately and get the best deal for the car than from local retailers. Prediction of the price of the car is based on certain parameters that were present in car price dataset by CarDekho from Kaggle which is obtained after applying some preprocessing techinque to get accurate result. This project was build using python with additional libraries such as Scikit-Learn, Numpy, Pandas, Flask and Pickle. You can look at the Documentation from Doc tab.
